### 🛠️ 问题解决记录

#### 遇到的问题
```
Error: spawnSync esbuild Unknown system error -88
```

#### 解决方案
使用以下命令成功解决：
```bash
# 1. 清理环境
rm -rf node_modules package-lock.json
npm cache clean --force

# 2. 重新安装
npm install --legacy-peer-deps
```

#### 原因分析
- macOS 安全限制导致 esbuild 二进制文件无法执行
- 使用 `--legacy-peer-deps` 可以避免依赖冲突
- 清理缓存确保干净的安装环境

### 📚 使用指南

#### 基础操作

1. **选择范围**
   - 左侧列表有 12 个预设范围
   - 点击任意范围立即加载

2. **选择手牌**
   - **单点**: 点击任意手牌格子
   - **滑动**: 按住鼠标拖动批量选择
   - **移动端**: 手指滑动选择

3. **查看概率**
   - 顶部显示已选手牌类型数
   - 显示实际组合数（对子6种·同色4种·不同色12种）
   - 实时显示出现概率
   - 自动计算每100次出现频率

4. **批量操作**
   - 全选/清空所有手牌
   - 反选当前选择
   - 按类型选择（对子/同色/不同色）

5. **自定义范围**
   - 点击左上角 "+" 创建新范围
   - 输入唯一名称
   - 开始选择手牌

6. **收藏功能**
   - 点击星标 ⭐ 收藏范围
   - 收藏的范围自动置顶

7. **导入导出**
   - 📥 下载图标：导出所有范围为 JSON
   - 📤 上传图标：导入备份文件

### 💡 使用技巧

1. **快速选择强牌**
   - 使用批量操作选择"所有对子"
   - 然后单独取消小对子（22-66）

2. **创建紧凶范围**
   - 选择 AA-TT（高对子）
   - 加上 AKs, AQs, AKo（大牌）

3. **创建LAG范围**
   - 使用 BTN Open 预设
   - 根据需要微调

4. **对比范围**
   - 创建多个范围
   - 通过概率对比强度

5. **保存常用范围**
   - 收藏常用的5-10个范围
   - 快速切换使用
