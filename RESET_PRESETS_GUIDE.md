# 预设范围重置功能指南

## 🔄 功能概述

预设范围重置功能允许您一键恢复所有预设范围到初始状态，非常适合以下场景：
- 不小心修改了预设范围
- 想要恢复官方推荐配置
- 清理测试数据

## 📍 功能位置

在"范围管理"面板的右上角工具栏，橙色的旋转图标：

```
┌─────────────────────────────────┐
│ 范围管理                         │
│                  [➕][🔄][📥][📤]│
│                   新  重  导  导 │
│                   建  置  出  入 │
└─────────────────────────────────┘
```

## 🎯 使用方法

### 步骤 1: 点击重置按钮
点击橙色的旋转图标 (🔄)

### 步骤 2: 确认操作
系统会弹出确认对话框：

```
┌──────────────────────────────────────┐
│ 确定要重置所有预设范围吗？            │
│                                       │
│ 这将恢复所有预设范围到初始状态。      │
│ 自定义范围不会受到影响。              │
│                                       │
│          [取消]    [确定]            │
└──────────────────────────────────────┘
```

### 步骤 3: 完成
点击"确定"后：
- ✅ 所有预设范围恢复到初始状态
- ✅ 显示"预设范围已重置"提示
- ✅ 自定义范围保持不变

## 🛡️ 安全保护机制

### 1. 二次确认
点击重置按钮后会弹出确认对话框，防止误操作。

### 2. 保留自定义范围
**完全不受影响**的内容：
- ✅ 所有自定义范围
- ✅ 自定义范围的手牌选择
- ✅ 自定义范围的收藏状态

### 3. 保留预设范围收藏
如果您收藏了某些预设范围，重置后收藏状态会保留。

**示例**:
```
重置前:
- ⭐ UTG Open (收藏)
- BTN Open
- ⭐ CO 3-Bet (收藏)

重置后:
- ⭐ UTG Open (收藏状态保留)
- BTN Open
- ⭐ CO 3-Bet (收藏状态保留)
```

### 4. 保持当前选中
如果您正在查看某个预设范围，重置后会自动切换到重置后的同一个范围。

## 📊 重置内容

### 会被重置的
- ✅ 预设范围的手牌选择
- ✅ 预设范围的手牌组合

### 不会被重置的
- ❌ 自定义范围（完全保留）
- ❌ 预设范围的收藏状态
- ❌ 范围的排序顺序

## ⚠️ 注意事项

### 1. 不可撤销
重置操作不可撤销，请在确认前仔细考虑。

**建议**: 如果不确定，可以先导出当前所有范围作为备份。

### 2. 自定义范围安全
自定义范围绝对安全，不会被重置功能影响。

### 3. 收藏状态保留
预设范围的收藏状态会自动保留，不需要重新收藏。

## 🔄 与其他功能的关系

### 重置 vs 删除
- **重置**: 恢复预设范围到初始状态
- **删除**: 只能删除自定义范围，预设范围不可删除

### 重置 vs 导入
- **重置**: 恢复内置的预设范围
- **导入**: 从文件导入自定义范围

### 重置 vs 清除所有数据
- **重置预设**: 只重置预设范围，保留自定义范围
- **清除所有数据**: 删除所有数据（包括自定义范围）

## 📝 技术细节

### 重置过程

```typescript
1. 识别自定义范围（isPreset: false）
2. 记录预设范围的收藏状态
3. 创建新的预设范围
4. 恢复收藏状态到新的预设范围
5. 合并自定义范围和新的预设范围
6. 保持当前选中的范围
7. 保存到本地存储
```

### 数据流

```
用户点击重置
    ↓
弹出确认对话框
    ↓
用户确认
    ↓
1. 筛选自定义范围
2. 记录收藏状态
    ↓
创建新的预设范围
    ↓
恢复收藏状态
    ↓
更新状态并保存
    ↓
显示成功提示
```

## 📞 常见问题

### Q: 重置后我的自定义范围会丢失吗？
**A**: 不会！自定义范围完全不受影响。

### Q: 重置后收藏的预设范围会取消收藏吗？
**A**: 不会！收藏状态会自动保留。

### Q: 如果误操作了怎么办？
**A**: 重置前建议先导出备份。误操作后可以从备份导入。

### Q: 重置会影响本地存储的其他数据吗？
**A**: 不会！只影响预设范围的手牌选择。

### Q: 可以只重置某一个预设范围吗？
**A**: 当前版本是全部重置。如需重置单个范围，可以：
1. 导出当前数据备份
2. 点击重置
3. 从备份中导入需要保留的其他范围
