# 德州扑克手牌范围计算器

一个功能强大的德州扑克手牌范围管理与概率计算工具，帮助玩家快速选择、管理不同场景下的手牌范围，实时计算并展示出现概率。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.2-blue)](https://www.typescriptlang.org/)
[![React](https://img.shields.io/badge/React-18.2-61dafb)](https://react.dev/)

## ✨ 核心特性

- 🎯 **可视化矩阵** - 13x13 直观展示所有 169 种手牌
- ⚡ **实时概率计算** - 准确计算出现频率（基于真实组合数）
- 📊 **12个预设范围** - 覆盖 UTG/HJ/CO/BTN/SB/BB Open 等常见场景
- 🎨 **灵活管理** - 自定义范围无限创建，收藏常用配置
- 💾 **数据安全** - 本地存储，支持导入导出，隐私保护
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 🚀 **高性能** - 纯前端实现，无网络延迟，可离线使用

## 🎯 核心功能

### 1. 手牌矩阵可视化
- 13x13 矩阵清晰展示所有手牌
- 对子、同色、不同色分区显示
- 点击或滑动批量选择手牌

### 2. 实时概率计算
- **已选手牌类型**: 10 种类型 / 169
- **实际组合数**: 68 种组合 / 1326
- **出现概率**: 5.13%

**计算公式**:
```
对子（如 AA）= 6 种组合
同色（如 AKs）= 4 种组合
不同色（如 AKo）= 12 种组合

概率 = 实际组合数 ÷ 1326 × 100%
```

### 3. 预设范围（12个）

| 范围名称 | 概率 | 说明 |
|---------|------|------|
| UTG Open | 15-20% | 最紧的开牌范围 |
| HJ Open | 20-25% | 中前位开牌 |
| CO Open | 25-30% | 关位开牌 |
| BTN Open | 40-50% | 最宽的开牌范围 |
| SB Open | 35-45% | 小盲位开牌 |
| BB vs BTN | 30-40% | 大盲防守 |
| CO 3-Bet | 8-12% | 三倍加注 |
| BTN Call 3-Bet | 15-20% | 跟注三倍加注 |
| 4-Bet | 4-6% | 四倍加注 |
| Call 4-Bet | 2-3% | 跟注四倍加注 |
| Flop Check | - | 翻牌圈过牌 |
| Flop C-Bet | - | 翻牌圈持续下注 |

### 4. 自定义范围
- 创建无限个自定义范围
- 编辑/删除/重命名
- 收藏常用范围（自动置顶）

### 5. 批量操作
- 全选/清空/反选
- 按类型批量选择（对子/同色/不同色）

### 6. 数据管理
- 导出范围为 JSON 文件
- 导入 JSON 范围文件
- 重置预设范围
- 本地自动保存
